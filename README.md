# CelebFace Evolution 🧑‍🎤📸

Koleksi foto **evolusi wajah tokoh terkenal dari tahun ke tahun** — dari muda sampai sekarang. Tiap tokoh punya satu folder sendiri.

## Struktur tiap tokoh
```
<nama-tokoh>/
├── README.md        # daftar foto + tahun + atribusi + lisensi
├── photos/          # foto berlisensi BEBAS (CC / Public Domain) — aman dipakai dengan atribusi
└── childhood/       # (opsional) foto masa kecil — bisa BERHAK CIPTA, cek README tokoh
```

## Cara menambah tokoh baru
1. Buat folder baru, mis. `cristiano-ronaldo/`, dengan subfolder `photos/`.
2. Isi foto berlisensi bebas (sumber bagus: [Wikimedia Commons](https://commons.wikimedia.org/) — filter lisensi CC/Public Domain).
3. Tulis `README.md` di folder tokoh: tabel tahun + fotografer + lisensi + sumber.
4. Tambahkan barisnya ke tabel **Daftar Tokoh** di atas.

## Lisensi & hak cipta ⚠️
- Foto di folder **`photos/`** tiap tokoh = **berlisensi bebas** (CC / Public Domain). Aman dipakai selama atribusinya disertakan (lihat README tokoh).
- Foto di folder **`childhood/`** atau yang bertanda **©** = **hak cipta pihak lain**. Hanya untuk referensi pribadi — **jangan dipublikasikan / dipakai komersial** tanpa lisensi resmi (mis. Getty Images / Alamy).
